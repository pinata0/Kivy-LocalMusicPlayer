# Kivy-LocalMusicPlayer
A music player based on local storage which made with Kivy

실행방법:

pip install kivy
pip install ffpyplayer
pip install mutagen
pip install PIL
pip install pandas
pip install numpy
pip install librosa
pip install transformers
pip install torch

main.py 실행
